## Introduction:
This is a project of CPSC 332 File structure and database class

## Prerequisites
Things you need to or recommend to have to run the project:
XAMPP to run local server
SQL, MySQL Workbench to import and open database scripts
Notepad, Notepad+, or Visual Studio Code to open PHP file

## Running the program
.Open XAMPP and start Apache and MySQL server.
.Open MySQL workbench and import database from doctoroffice.sql
.Open your web browser and type in localhost/Hien_Nguyen_CPSC_332_Project/index.php
